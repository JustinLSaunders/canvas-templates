The exam(s) in this course will be administered with the assistance of an automated remote proctoring solution, Proctorio.

Proctorio will monitor and record your webcam video, audio, keystrokes, web traffic, and desktop. It will automatically flag behavior that may compromise academic integrity. The report generated by Proctorio will be reviewed by your instructor, TA and additional support staff.

To guarantee that technology is not a prohibitive barrier to your exam experience, we have created this preparation activity to walk you through the setup process for Proctorio. Make sure you complete this activity well in advance of your upcoming exam.

Note: If the system asks for an access code when you try to start a Proctorio quiz, this is a technical glitch. Go back to the course home page and restart the quiz.

Requirements

  1. Install Google Chrome. If you are using another browser—such as Firefox, Safari, or Internet Explorer—you will need to exit and restart in Chrome. (skip this step if you have Google Chrome installed):
    • Google Chrome for Windows (Links to an external site.)
    • Google Chrome for Mac (Links to an external site.)
  2. Install the Proctorio Extension (Links to an external site.)
    1. Click the link under Install Proctorio Chrome Extension button to begin the download process.
    2. You will receive confirmation when the Proctorio extension is successfully downloaded and added to Chrome.
    3. You may need to enable the Proctorio extension to run in incognito mode.
      1. Go to chrome://extensions.
      2. Find Proctorio, and check the Allow in incognito box.
  3. Finally, to make certain that Proctorio will work for you on the day of the exam, complete this brief quiz by clicking Take the Survey below.

If you have any additional questions about the proctoring experience, Proctorio has a 24/7 toll-free number you can call: 866.948.9248. Additionally, you can, at any time before or during the exam, initiative a live chat with a support personnel by clicking on the Secure Exam Proctorio shield icon in the Google Chrome address bar, and selecting the Live Chat button.
